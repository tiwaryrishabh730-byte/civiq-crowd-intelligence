**CIVIQ (Crowd Intelligence & Vision)**

Transforming Urban Mobility through Crowdsourced Live Intelligence.

CIVIQ is a next-generation urban crowd monitoring application designed to solve the problem of unpredictable waiting times in public spaces. By leveraging real-time user contributions and data-centric design, it provides citizens with live insights into crowd density across the city.

**Live Demo**

Check it out here: https://civiq-crowd-intelligence-du92.vercel.app/

 **Features**
 
Live Overwatch: A real-time map interface to monitor crowd density across various urban hotspots.

Crowdsourced Reports: Users can contribute live status updates for public spaces (Bus stops, Railway stations, Malls, etc.).

Predictive Waiting Times: Uses historical and real-time data to estimate wait durations.

Data-Centric Minimalism: A modern, distraction-free "Swiss-style" UI for quick information consumption.

Responsive Experience: Built with a mobile-first approach for users on the move.

**Tech Stack**

Frontend: **Next.js 14 (App Router), React, TypeScript**

Styling: **Tailwind CSS**

Animations: **Framer Motion**

Backend & Database: **Firebase (Firestore & Auth)**

Maps: **React-Leaflet**

Deployment: **Vercel**

**Architecture**

The app follows a modern serverless architecture:

Frontend: Fetches real-time snapshots from Firestore.

State Management: Reactive UI updates based on crowd reporting triggers.

Security: Granular Firestore Security Rules to prevent unauthorized data manipulation.
